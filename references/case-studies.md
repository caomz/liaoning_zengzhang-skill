## 破局点公式经典案例

### 案例1：微信支付 vs 支付宝

| 维度 | 微信支付的破局 |
|------|--------------|
| 相对广谱 | ✓ 所有微信用户（10亿+） |
| 高频 | ✓ 红包场景（极高频） |
| 体验可控 | ✓ 微信内闭环，体验完全可控 |
| 破局点 | 微信红包——用社交场景切入支付 |

### 案例2：拼多多的破局

| 维度 | 拼多多的破局 |
|------|------------|
| 相对广谱 | ✓ 价格敏感用户（广大下沉市场） |
| 高频 | ✓ 日常消费品（极高频） |
| 体验可控 | ⚠️ 品质问题是挑战 |
| 破局点 | 社交拼团——用微信社交裂变降低获客成本 |
