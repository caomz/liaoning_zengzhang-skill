# 梁宁·增长思维30讲技能

> 增长能力 = 持续作出正确决定的能力

## 概述

本技能基于梁宁《增长思维30讲》课程开发，提供完整的企业增长诊断框架。通过六大模块（增长地图、机会识别、模式设计、组织诊断、借势策略、用户深化）和24个核心模型，帮助用户诊断企业/产品/组织的增长问题，并给出结构化的行动建议。

## 使用方式

### 触发词

```
「如何增长」「判断阶段」「找破局点」「商业模式分析」
「组织诊断」「借势策略」「用户深化」「跨越周期」
「我在哪里」「机会在哪」「如何扩张」「团队诊断」
```

### 快速诊断

适用于用户有一个明确的小问题，30秒-2分钟给出诊断：

```
[你的问题] → 问题分类 → 调用1-2个模型 → 给出1-2个关键建议
```

### 深度咨询

适用于战略规划、全面诊断、重大决策，15-30分钟给出完整报告：

```
[你的情况] → 六大模块全面扫描 → 识别核心瓶颈 → 优先级排序 → 分阶段行动计划
```

## 安装

本技能为纯文本技能，无需安装，直接在 Claude Code 中使用触发词激活。

### 工具依赖（可选）

如需使用增长计算器：

```bash
pip install -r requirements.txt
# 或直接运行
python tools/growth_calculator.py
```

## 使用示例

**Q**：我的产品刚上线，用户增长很慢，怎么办？

**A**：启动快速诊断——
1. 定位模块：增长地图（草莽阶段）+ 机会（破局点）
2. 调用模型：破局点公式
3. 关键问题：你找到"相对广谱+高频+体验可控"的切入点了吗？
4. 建议方向：先集中资源在一个破局点上，不要分散

**Q**：我们要做战略规划，请全面分析

**A**：启动深度咨询——
1. 六大模块全面扫描
2. 识别核心瓶颈
3. 优先级排序
4. 制定一横一纵行动计划
